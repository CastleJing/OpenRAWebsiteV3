Community

OpenRA is a game created by the community, for the community. You can contribute in a variety of ways, from giving help to the dev team, to finding and reporting bugs, to just playing the game!

Connect With Us on Social Media
We're active on Reddit, Twitter, and Facebook. Follow us there to get the latest news on features and releases!

Reddit Twitter Facebook YouTube

Help Develop The Game
If you run into problems, your first stop should be the OpenRA FAQ. If you don't find what you need in the FAQ, file a bug on the OpenRA Bug Tracker.

  • OpenRA Bug Tracker
