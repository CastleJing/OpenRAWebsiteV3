我应该选择哪一个版本？

OpenRA提供两个可选版本，分别是测试版（Playtest）和发布版（Release），你可以根据自己的需求自行选择。

  • 正式版足以满足绝大部分玩家的需求；
  • 测试版作为未来OpenRA正式版的预览，更适合那些想要抢先体验，或者想要协助我们完善OpenRA的玩家们。

游戏素材与权属说明

OpenRA（游戏及游戏引擎）是基于GPL3协议的免费开源软件，的免费开源软件；然而OpenRA的模组必须依赖原始游戏的素材文件才能正常运行。上述素材文件的使用均遵循EA发布的命令与征服系列模组说明书（C&C Franchise Modding Guidelines）。

请注意，OpenRA并不持有上述“原始游戏之素材文件”的权属，您必须从合法且正当的来源自行获取运行OpenRA所必须的上述文件。

社区维护包

OpenRA也可以使用Linux发行版或其他第三方提供的软件包进行安装。这些软件包将提供与您的系统更好的集成，但可能并不总是提供最新版本。有关任何打包问题，请联系下游存储库维护人员。

请注意，OpenRA官方不负责此类社群平台的维护工作；在新版本发布后，此类下游平台可能会出现一些发布延迟

Flatpak

从Flathub获得稳定发布版及可玩性测试版：

$ flatpak install --from https://flathub.org/repo/appstream/net.openra.OpenRA.flatpakref $ flatpak install --from https://flathub.org/beta-repo/appstream/net.openra.OpenRA.flatpakref

请注意，Flatpak沙盒会干扰官方和社区mod之间的mod切换功能。我们建议希望使用此功能的玩家运行AppImage版本。

Snap

从Snap Store获得稳定发布版：

$ snap install openra

Arch Linux

从官方Arch 仓库获得稳定发布版：

openSUSE Linux
从openSUSE Build Service repository获得稳定发布版及可玩性测试版：

Solus
